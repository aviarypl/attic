Wymagania Thunderbirda

Sprzętowe i systemowe wymagania Thunderbirda przedstawiają się następująco:

Microsoft Windows

Wymagania sprzętowe:

  • procesor Pentium 233 MHz
    (lub zgodny; zalecane: 500MHz lub wyżej)
  • 64 MB pamięci RAM
    (zalecane: 128 MB lub więcej)
  • 52 MB wolnego miejsca na dysku twardym

Systemy operacyjne:

  • Windows 98
  • Windows 98SE
  • Windows ME
  • Windows NT 4.0
  • Windows 2000 (zalecany)
  • Windows XP (zalecany)
  • Windows Server 2003 (zalecany)

GNU/Linux

Wymagania sprzętowe:

  • procesor Pentium 233 MHz
    (lub zgodny; zalecane: 500MHz lub szybszy)
  • 64 MB pamięci RAM
    (zalecane: 128 MB lub więcej)
  • 52 MB wolnego miejsca na dysku twardym

System operacyjny:

  • jądro Linux w wersji 2.2.14 lub wyższej
  • biblioteka glibc 2.3.2 lub nowsza
  • biblioteka gtk+ 2.0.x lub nowsza
  • XFree86-3.3.6 lub nowsze

Dystrybucje GNU/Linuksa:

  • Red Hat Linux 7.0 lub nowszy
  • Fedora Core 1 lub nowsza
  • Aurox Linux 8.0 lub nowszy
  • SuSE Linux 8.0 lub nowszy
  • Slackware Linux 7.0 lub nowszy
  • Linux-Mandrake 8.2 lub nowszy
  • Gentoo Linux 1.4 lub nowszy
  • PLD 1.0 lub nowsza
  • Sun Java Desktop System w dowolnej wersji
  • i dowolna inna dystrybucja spełniająca w/w wymagania

Mac OS X

Wymagania sprzętowe:

  • procesor PowerPC G3
    (zalecany: PowerPC G4 667MHz lub szybszy)
  • 128 MB pamięci RAM
    (zalecane: 256 MB lub więcej)
  • 72 MB wolnego miejsca na dysku twardym

Systemy operacyjne:

  • Mac OS X 10.2.x lub nowszy
