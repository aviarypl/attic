Comment 5 Stefan Plewako :stef 2009-10-31 08:31:17 CET

Gdybyś mógł ustalić wartości które będą wystarczające przy pomocy poniższej linijki w userChrome.css:

#accountManager { width: 107ch !important; height: 49em !important; }
