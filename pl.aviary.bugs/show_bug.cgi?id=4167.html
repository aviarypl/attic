Pierwsza wersja przetlumaczonego pliku util.pm

Funkcji w bloku Utility Functions i Setup Functions nie ruszalem wogole - wydaje mi sie ze tam nie ma nic do tlumaczenia.

Zajalem sie Helpers for Selenium Scripts

Zastanawiam sie czy mam tlumaczyc rowniez takie cos np:

sub edit_product {
    my ($sel, $product, $classification) = @_;

    $classification ||= "Unclassified";
    go_to_admin($sel);
    $sel->click_ok("link=Produkty", undef, "Przejdź do strony produktów");
    $sel->wait_for_page_to_load(WAIT_TIME);
    my $title = $sel->get_title();
    if ($title eq "Select Classification") {
        ok(1, "More than one enterable classification available. Display them in a list");
        $sel->click_ok("link=$classification", undef, "Choose $classification");
        $sel->wait_for_page_to_load(WAIT_TIME);
    }
    else {
        $sel->title_is("Modyfikowanie produktów", "Display the list of enterable products");
    }
    $sel->click_ok("link=$product", undef, "Choose $product");
    $sel->wait_for_page_to_load(WAIT_TIME);
    $sel->title_is("Modyfikowanie produktu '$product'", "Display properties of $product");
}

Generalnie rozchodzi sie o ten ostatni parametr w wywolaniach funkcji selenium - to wyglada na zwykly komunikat diagnostyczny o arbitralnej tresci ustalonej przez autora angielskiej wersji testow. Na teraz przetlumaczylem wiekszosc z nich ale mysle ze usune te moje tlumaczenia i zostawie tak jak jest domyslnie, aby nie wprowadzac wiekszego zamieszania.
Mysle ze ten ostatni parametr nie ma wplywu na samo funkcjonowanie testow a tylko na to co pojawi sie w konsoli.
Najwazniejsze jest aby np. title_is("tytul") odnosil sie do tego co stoi w polskim UI.

Na koniec - gdzie znajde jakas sensowna dokumentacje tych wszystkich funkcji/metod selenium w klarownej formie? Wiele z nich jest przeladowanych i przyjmuje zmienna ilosc parametrow - fajnie byloby wiedziec co jest co dokladnie. Probowalem googlowac aczkolwiek rezultaty nie uznalem za zadowalajace.
