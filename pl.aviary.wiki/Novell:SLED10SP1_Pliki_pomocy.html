 3.1. To Open a File
 To open a File, perform the following steps:
   1. Choose File → Open.
   2. In the Load file dialog, select the file you want to open.

 3.1 Otwieranie pliku (rzeczownikowo)
 Aby otworzyć plik, należy: (czynność)
   1. Z menu programu wybrać Plik → Otwórz,
   2. W oknie dialogowym, zaznaczyć żądany plik ...
